APA ITU SQL?

SQL adalah suatu bahasa komputer standar yang digunakan untuk mengakses dan melakukan manipulasi suatu sistem database. Saat ini SQL dapat dijumpai pada berbagai platfrom, dari mikrokomputer hinggan mainframe. SQL dapat digunakan baik secara berdiri sendiri maupun dilekatkan pada bahasa-bahasa lain seperti C dan Delphi. Dan menjadi bagian dari sejumlah DBMS, seperti Oracle, Sybase, MySQL, dan Informix.

KENAPA SQL PERLU DIPELAJARI?

Di dunia virtual, aplikasi dan database web modern memiliki peran dan fungsi yang sangat penting. Karena pengembangan web saat ini begitu dinamis, aplikasi-aplikasi yang belum sempurna pun membutuhkan beberapa mekanisme seperti menyimpan, mengambil kembali, dan memodifikasi data. Sejak kebutuhan akan database terus meningkat, popularitas Relational Database Management Systems (RDBMS) pun semakin melonjak. Dan salah satu dari sistem ini adalah SQL, sistem database pertama yang dibuat pada tahun 90-an. Walaupun SQL tidak menjadi RDBMS yang paling populer karena masih ada Oracle dan Postgres, SQL saat ini masih relevan bahkan dominan digunakan dalam manajemen database terutama untuk pengembangan aplikasi kecil hingga medium dan perusahaan start-up karena bersifat open-source namun masih relevan digunakan saat ini.
