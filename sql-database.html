DATABASE

Database (basis data) adalah kumpulan data yang disimpan secara sistematis di dalam komputer yang dapat diolah atau dimanipulasi menggunakan perangkat lunak (program aplikasi) untuk menghasilkan informasi. Untuk membuat database dengan SQL, gunakan sintaks berikut:


CREATE DATABASE nama_database

Jika kamu ingin membuat database SQL “BasDatTest”, maka sintaks yang perlu digunakan adalah:


CREATE DATABASE BasDatTest

NOTE :

sintaks pada SQL tidak case sensitive, berarti penulisan create database dan CREATE DATABASE tidak akan mengubah hasil.

Sedangkan untuk menghapus database dengan SQL, gunakan sintaks berikut:


DROP DATABASE nama_database

Jika kamu ingin menghapus database SQL “BasDatTest”, maka sintaks yang perlu digunakan adalah:


DROP DATABASE BasDatTest
