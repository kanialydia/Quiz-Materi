WHERE

berfungsi untuk menentukan suatu pilihan atau kondisi tertentu.


SELECT * FROM namatabel where namakolom = “kondisi”

Contoh:
Menampilkan data guru yang beralamat hanya di Bogor.


SELECT * FROM Data_Guru WHERE gu_alamat = ‘Bogor’



AND

AND digunakan untuk memilih nilai yang memiliki kondisi sama antara kondisi1 dan kondisi2.


SELECT * FROM namatabel where kondisi1=’..’ AND kondisi2 = ‘…’;

Contoh:
Menampilkan data guru yang hanya beralamat di Bogor dan hanya mengajar Fisika.


SELECT * FROM Data_Guru WHERE gu_alamat = ‘Bogor’ AND gu_matpel = ‘Fisika’;



OR

OR ini berfungsi untuk kondisi yang ingin menampilkan data dengan keadaan atau.


SELECT * FROM namatabel where kondisi1=’..’ OR kondisi2 = ‘…’;

Contoh:
Menampilkan data guru yang hanya beralamat di Bogor atau mengajar Kimia.


SELECT * FROM Data_Guru WHERE gu_alamat = ‘Bogor’ OR gu_matpel = ‘Kimia’;



NOT

NOT berfungsi sebagai kondisi yang tidak kita kehendaki.


SELECT * FROM namatabel where kondisi1=’..’ NOT kondisi2 = ‘…’;

Contoh:
Menampilkan data guru yang beralamat di Bogor tidak mengajar Kimia.


SELECT * FROM Data_Guru WHERE gu_alamat = ‘Bogor’ NOT gu_matpel = ‘Kimia’;
